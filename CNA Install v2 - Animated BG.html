Installa su smartphone

iPhone & iPad Safari

1
Apri il link dell'app con Safari — non Chrome o altri browser
2
Tocca il pulsante ⬆ Condividi nella barra in basso
3
Scorri l'elenco e tocca "Aggiungi a schermata Home"
4
Tocca "Aggiungi" in alto a destra per confermare
5
L'icona CNA compare sulla Home — l'app si apre a schermo intero ✓

Android Chrome

1
Apri il link dell'app con Chrome
2
Tocca i ⋮ tre punti in alto a destra
3
Seleziona "Aggiungi a schermata Home" o "Installa app"
4
Tocca "Aggiungi" per confermare — icona CNA installata ✓
Installa su computer

Windows Chrome / Edge

1
Apri il link dell'app in Chrome o Microsoft Edge
2
Clicca i ⋮ tre punti in alto a destra
3
Seleziona "Installa ERP CNA di Roma" (o "Installa Signature")
4
Clicca "Installa" — l'app viene aggiunta al desktop e al menu Start ✓

macOS Chrome / Safari

1
Apri il link in Chrome o Safari
2
Chrome: clicca i ⋮ tre punti → "Installa app"
3
Safari: menu File → "Aggiungi al Dock" (macOS Sonoma+)
4
L'app si apre in una finestra dedicata, senza barra del browser ✓
Credenziali di accesso

Per accedere alle app sono necessarie le credenziali fornite dall'amministratore CNA. Una volta installata, l'app funziona come un programma nativo — icona nel desktop, nessuna barra del browser. In caso di problemi contatta il responsabile IT.
